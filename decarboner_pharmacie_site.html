À propos du site

Ce site a été pensé pour être simple à modifier : tout le contenu est dans un seul fichier HTML. Pour ajouter un nouveau projet, une nouvelle méthode ou une nouvelle publication, il suffit de dupliquer une entrée dans les tableaux JavaScript situés en bas du fichier.

Hébergement conseillé : GitHub Pages, Netlify ou Cloudflare Pages. Les trois options ont une formule gratuite suffisante pour ce type de site.
